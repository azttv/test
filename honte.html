Comment rejoindre (Java Edition)

  1. Ouvrez Minecraft Java Edition (version 1.21.4 ou supérieure)
  2. Depuis le menu principal, cliquez sur "Multijoueur"
  3. Cliquez sur "Ajouter un serveur"
  4. Dans le champ "Adresse du serveur", saisissez play.true-vanilla.fr
  5. Cliquez sur "Terminé" puis sélectionnez le serveur dans la liste
  6. Cliquez sur "Rejoindre le serveur"
play.true-vanilla.fr

Comment rejoindre (Bedrock Edition)

  1. Ouvrez Minecraft Bedrock Edition
  2. Allez dans l'onglet "Serveurs"
  3. Faites défiler jusqu'en bas et cliquez sur "Ajouter un serveur"
  4. Remplissez les champs suivants:
    • Nom du serveur: True Vanilla
    • Adresse du serveur: play.true-vanilla.fr
    • Port: 50327
  5. Appuyez sur "Enregistrer"
  6. Sélectionnez "True Vanilla" dans votre liste de serveurs
Note: True Vanilla est compatible avec toutes les plateformes Bedrock (mobile, console, Windows 10/11).
